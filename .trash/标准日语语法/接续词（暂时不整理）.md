## 1. 原因 → 结果 / 结论

| 接续词   | 用法                    |
| ----- | --------------------- |
| だから   | 最普通的“所以”。口语常用，可以接主观判断 |
| そのため  | 较书面，强调客观原因导致结果        |
| したがって | 正式推论，“因此可得出……”        |
| よって   | 更硬，常用于证明、法律、论文        |
| それゆえ  | 书面语“因此”，比「だから」硬       |
| ゆえに   | “故，因此”，更文学、哲学、论文感     |

内部区别：

「だから」最口语，日常说话最常用。

雨が降っている。だから、行かない。  
下雨了。所以不去。

「そのため」更适合说明客观事实。

台風が近づいている。そのため、試合は中止になった。  
台风接近。因此比赛取消了。

「したがって」和「よって」更像逻辑推论。

この方法には問題がある。したがって、採用できない。  
这个方法有问题。因此不能采用。

AはBに等しい。よって、Cも成立する。  
A等于B。因此C也成立。

「それゆえ」「ゆえに」更硬，更有书面、文学、哲学感。

人は有限である。ゆえに、時間は貴重だ。  
人是有限的。因此，时间很宝贵。

简单顺序：

だから ＜ そのため ＜ したがって ＜ よって  
口语 → 书面 → 正式推论 → 极正式证明

## 2. 事情发展 → 后续

|接续词|用法|
|---|---|
|それで|前面的事情自然引出后面的事情|
|で|「それで」的口语压缩版，讲故事、聊天常用|

「それで」偏“事情这样发展，于是……”。

財布を忘れた。それで、家に戻った。  
忘带钱包了。于是回家了。

「で」更口语，更像聊天时轻轻带一句“然后”。

昨日駅に行った。で、友達に会った。  
昨天去了车站。然后见到了朋友。

区别：

それで：比较完整，叙述、说明都能用。  
で：非常口语，正式文章和报告里少用。

这篇文章也提到，日语母语者在口语叙述中常用「で」「それで」，而学习者常常不太会区分它们。

## 3. 时间顺序推进

| 接续词  | 用法              |
| ---- | --------------- |
| それから | 单纯表示时间上的“然后、接着” |
| そして  | 连接后续事件，后面往往更有分量 |

「それから」只是按时间顺序继续说。

朝ご飯を食べた。それから、学校へ行った。  
吃了早饭。然后去了学校。

「そして」不只是“然后”，后面常常接更重要、总结性、决定性的内容。

彼は十年努力した。そして、ついに成功した。  
他努力了十年。然后终于成功了。

所以这句不太自然：

朝起きた。そして、歯を磨いた。そして、ご飯を食べた。そして、学校へ行った。

更自然：

朝起きて、歯を磨いて、ご飯を食べて、それから学校へ行った。

文章里也指出，中国学习者容易过度使用「そして」，而日语母语者口语里并不常这样用。

## 4. 前项发生 → 后项立刻出现

|接续词|用法|
|---|---|
|すると|前面一发生，后面马上发生某事，或发现某事|

「すると」常用于故事、小说、发现。

ドアを開けた。すると、猫が入ってきた。  
打开门。于是猫进来了。

箱を開けた。すると、中に手紙が入っていた。  
打开箱子。结果里面有封信。

它和「それから」不同：

ドアを開けた。それから、猫が入ってきた。  
只是“然后猫进来了”。

ドアを開けた。すると、猫が入ってきた。  
有“门一开，猫就进来了”的感觉。

---

## 5. 出现情况 → 采取行动

|接续词|用法|
|---|---|
|そこで|前面出现问题、状况，后面主动采取措施|

「そこで」后面常接人的行动、判断、决定。

道に迷った。そこで、近くの人に聞いた。  
迷路了。于是问了附近的人。

問題が起きた。そこで、会議を開くことにした。  
出了问题。于是决定开会。

和「それで」的区别很重要：

電車が止まった。それで、遅刻した。  
电车停了。所以迟到了。  
→ 自然结果。

電車が止まった。そこで、タクシーに乗った。  
电车停了。于是坐了出租车。  
→ 采取行动。

## 6. 这样一来 / 最终形成结果

|接续词|用法|
|---|---|
|そうすると|这样一来，就会……|
|その結果|结果，最终出现了某结果|
|こうして|经过前面过程，就这样形成结果|

「そうすると」常用于说明方法、条件、操作后的结果。

ここを押す。そうすると、画面が変わる。  
按这里。这样一来，画面就会变。

「その結果」强调“结果变成了……”。

毎日練習した。その結果、試合に勝った。  
每天练习。结果赢了比赛。

「こうして」是总结前面一连串过程。

彼は毎日練習した。何度も失敗した。それでも続けた。こうして、プロになった。  
他每天练习，失败很多次，但仍然坚持。就这样，他成了职业选手。

区别：

そうすると：这样做 → 会怎样。  
その結果：最后结果是怎样。  
こうして：经过这些过程 → 就这样形成结果。


| 级别  | 接续词                    |
| --- | ---------------------- |
| 一级  | でも、けど、だけど、しかし          |
| 二级  | ただ、それでも、ところが、けれども、とはいえ |
| 三级  | それなのに、それにしても、もっとも      |
## 1. 普通转折

|接续词|区别|
|---|---|
|でも|最普通、最常用的口语“但是”|
|けど|很口语，语气轻，也常用于句尾|
|だけど|比「けど」更像独立接续词，“不过、可是”|
|しかし|较正式，文章、说明、演讲中常用|
|けれども|比「けど」正式、柔和，比「しかし」没那么硬|

例：

行きたい。でも、時間がない。  
想去。但是没时间。

面白い。だけど、少し難しい。  
很有趣。不过有点难。

努力した。しかし、結果は出なかった。  
努力了。但是没有结果。

この本は難しい。けれども、読む価値はある。  
这本书很难。不过有读的价值。

简单区别：

|想表达|用|
|---|---|
|日常“但是”|でも|
|很口语的“不过”|けど、だけど|
|正式“但是”|しかし|
|柔和正式的“不过”|けれども|

## 2. 意外 / 反预期转折

|接续词|区别|
|---|---|
|ところが|后面的结果和预想相反，带“结果却……”|
|それなのに|明明前面是那样，后面却这样；带不满、遗憾、惊讶|

例：

彼は来ると言っていた。ところが、来なかった。  
他说会来。然而结果没来。

一生懸命勉強した。それなのに、試験に落ちた。  
拼命学习了。可是考试还是没过。

区别：

「ところが」更像客观叙述“结果却”。  
「それなのに」更带情绪，“明明如此，可是……”。

## 3. 让步转折

|接续词|区别|
|---|---|
|それでも|承认前面的困难、问题，但后面仍然成立|

例：

雨が降っていた。それでも、彼は出かけた。  
下着雨。即便如此，他还是出门了。

失敗した。それでも、あきらめなかった。  
失败了。即便如此，也没有放弃。

和「それなのに」的区别：

|接续词|重点|
|---|---|
|それなのに|明明 A，却发生了不该发生的 B|
|それでも|即使 A，仍然坚持 / 仍然发生 B|

例：

勉強した。それなのに、落ちた。  
学了。可是却没考过。  
→ 反预期、遗憾。

難しい。それでも、続ける。  
很难。即便如此还要继续。  
→ 让步、坚持。

## 4. 补充限制

|接续词|区别|
|---|---|
|ただ|最常用，“不过有一点……”|
|もっとも|“不过话说回来……”，偏书面，常补充限制条件|

例：

この店は安い。ただ、駅から遠い。  
这家店便宜。不过离车站远。

彼は優秀だ。もっとも、経験はまだ少ない。  
他很优秀。不过话说回来，经验还少。

区别：

「ただ」很常用，直接补充缺点、限制、例外。  
「もっとも」更像文章里补一句修正，语气更书面、更冷静。

## 5. 补充修正 / 评价转折

|接续词|区别|
|---|---|
|とはいえ|承认前文，但后面降低、限制、修正前文|
|それにしても|即便考虑到前文，仍然觉得后文程度很强；常带评价、感叹|

例：

日本語は少し読めるようになった。とはいえ、小説はまだ難しい。  
日语稍微能读了。话虽如此，小说还是难。

雨が降っている。それにしても、人が多いね。  
在下雨。话说回来，人还是很多啊。

区别：

「とはいえ」= 虽然这么说，但实际还不能说到那个程度。  
「それにしても」= 即便如此，也太……了吧 / 还是很……啊。
